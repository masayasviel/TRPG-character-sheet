# TRPG キャラクターシート

# Install
```
npm install
```

# Requirement

- node v14.3.0
- npm v6.14.5
- @vue/cli v4.5.6

# Usage

## Compiles and hot-reloads for development
```
npm run serve
```

## Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
